被监控主机设置

cacti监控方式：snmp
被监控主机 也需要支持：snmp

配置被监控的Linux客户机

说明：要使用CactiEZ监控一台Linux主机，需要在被监控的主机上安装net-snmp等相关的软件包；
同时需要开启防火墙UDP161端口

  1. 开启防火墙UDP161端口

    • vi /etc/sysconfig/iptables #编辑防火墙配置: -A INPUT -m state --state NEW -m udp -p udp --dport 161 -j ACCEPT
    • /etc/init.d/iptables restart #重启防火墙使配置生效

  2. 配置snmp

    • cp /etc/snmp/snmpd.conf /etc/snmp/snmpd.confbak #备份配置文件

    • vi snmpd.conf #添加下面代码

    • service snmpd restart #重启snmp

    • netstat -nlup |grep “:161” #检查snmp服务器是否运行，出现下面输出结果，说明snmp运行正常
      udp 0 0 0.0.0.0:161 0.0.0.0:* 3265/snmpd
      监控主机操作：
      snmpwalk -v 2c -c public 192.168.0.117（若timeout，关闭被监控linux主机的防火墙）
      -v版本
      -c 以哪个组的身份
      链接ip地址；
      验证是否监控成功，5s中内完成，则失败；
