###系统相关

  • 查看用户名：hostname
  • 查看用户相关： uname -a
  • 版本号：uname -r 主+次+派生+修正
  • 次生版本：奇数，测试版本；偶数，稳定版本
  • echo “hello” -S “from root”|mail tom@localhost
  • cat /etc/shells 查看shell
  • passwd -d 删除用户密码
  • passwd -l 锁定用户

###如何切界面
linux 分 图形界面 文本界面
linux有7个运行级别
init 命令

  • 文本界面 init 1 2 3 4
    • init 1 单用户界面 安全模式 不支持网络
    • init 2 不支持网络的多用户界面
    • init 3 完全 多用户 文本界面 一般我们工作在这个模式
    • init 4 未定义
    • init 5 图形界面
    • init 0 关机
    • init 6 重启

###linux 关机 重启 注销
关机命令 通用命令 halt | init 0 shutdown -h {+N|now}
重启命令 reboot | init 6 shutdown -r {+N|now}
注销命令 logout exit

###常用命令

  • 进入文件夹的命令 cd
  • cd - 返回上次所在目录 cd .. 返回上一级目录
  • 定位自己当前位置 pwd
  • 清屏 clear ctrl+l
  • 列出某个文件夹下的子目录或文件 ls
    • ls -l 列出详细信息 列表形式列出
    • -h 列出文件大小的时候 以 K M G单位显示
    • d 显示目录的详细信息
    • -i 显示该文件的i节点 对应的系统中只有一个文件

  • cp 复制命令 可以复制改名；

    • cp -a 防止权限改变
    • cp -r 复制目录时使用

  • 列出上次使用命令空格后面的内容 alt+.

  • 帮助 man 例子：man ls

###别名：
设置别名：alias
删除别名：unalias

例如: alias ls=’cd /‘ ls命令即被修改成cd /命令；
unalias ls ls即恢复正常；

###文件和文件夹的管理
注意:linux设计思想 在linux 一切都是文件
在linux中 所有设备都是一个文件
例子：/dev/cdrom 代表的就是光驱

注意 linux 系统中 根下文件夹介绍

  • /bin 普通用户和管理员可以使用的命令
  • /sbin 管理员可以使用的命令
  • /dev 设备文件 光驱 硬盘 等
  • /etc 配置文件
  • /home 普通用户的家目录
  • /media 自动挂载目录 /mnt 手动挂载目录
  • /lib /lib64 库文件
  • /opt 三方程序安装路径
  • /root 管理员的家目录
  • /tmp 临时文件存放目录
  • /var 日志存放目录
  • /proc 内存系统

###文件夹的管理

  1. 增 mkdir 文件夹名字
    • -p 递归创建
      • -v 显示创建结果
  2. 删 空文件夹 rmdir 不为空的目录 rm -r 目录名
  3. 改 mv 改名 移动
    • mv 源文件名 目标文件名
    • 如果目标文件名不是一个目录 执行改名
    • 如果目标文件名是一个目录 执行移动
  1. 查 ls -l

###文件的管理

  1. 增 touch filename
  2. 删 rm -r 递归删除 -f 强制直接删除 注意：rm 删除不走回收站
  3. 改 mv 改名 移动
    mv 源文件名 目标文件名
    如果目标文件名不是一个目录 执行改名
  4. 查 cat 查看文件内容

编辑文件的命令 图形界面 gedit 文本界面 vim

###文件的基本权限

-rw-r--r--. 1 root root 62 file1
r w write x -表示 没有权限

####文件的类型

  • 普通文件
  • d 目录
  • l 链接 快捷方式
  • b 设备文件
  • c 字符型设备
  • p 管道文件

####权限修改
chmod

  • a all
  • u user
  • g group
  • o other
  • +添加权限
  • -删除权限

权限

  • r read
  • w write
  • x execute
    • 没有权限

修改权限方法:

  1. chmod u+x,g-w,o-rwx file1
  2. 数字加权法: 所以一类用户对它如果有所有权限 数字就是4+2+1=7 7 7 7
    • r 4
    • w 2
    • x 1

chmod 664 file1
rw-rw-r–

###所有者 所属组修改

所有者 chown chown 新的所有者 文件或目录名
所属组 chgrp chgrp 新组名 文件或目录名

命令选项 如果对文件夹进行修改的时候 希望子目录也跟着修改 请用-R来递归设置 适用于 chmod chown chgrp

如果用户名和组都想改
chown 新用户名.新组名 文件名
